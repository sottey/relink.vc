# add command
Adds a link

Adding a link is very simple. You just need to provide the link and, optionally, tags for that link:

`relinkvc add https://github.com `

## Usage
`redovc add <link>`

## Aliases
  add, a, new

## Examples
`redovc add "https://github.com"`
`redovc new https://github.com`

## Flags
`-h, --help   help for add`
